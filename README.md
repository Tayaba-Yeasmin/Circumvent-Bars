# Circumvent Bars – Cross-Platform Arcade Game

## Overview
Circumvent Bars is a cross-platform arcade-style game developed using Unity and C#. The objective of the game is to control a ball, avoid obstacles, and reach the goal across multiple levels. The game is designed for both PC and mobile platforms, offering a simple yet engaging gameplay experience.

---

## Features
- Multi-level gameplay (4 levels with increasing difficulty)
- Obstacle avoidance mechanics
- Score tracking system
- Game over and level completion screens
- Instruction page for user guidance
- Menu navigation system (Start, Levels, Instructions, Quit)
- Cross-platform support (PC and Mobile)

---

## Gameplay Mechanics
- Player controls a ball and navigates through obstacles
- Objective is to reach the endpoint without collision
- Controls:
  - PC:
    - Move Left: D
    - Move Right: J
    - Jump: T
  - Mobile:
    - On-screen buttons (<, >, ^)
- If the ball falls or hits an obstacle:
  - Game restarts or ends
- Completing a level unlocks the next level
- Completing all levels leads to a "Game Won" screen

## Game Flow
1. Launch game → Start screen  
2. Navigate to menu  
3. Select level or view instructions  
4. Play and complete levels  
5. Progress through all levels to win
